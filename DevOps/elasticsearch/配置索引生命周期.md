
> 先做一个生命周期策略

```bash

PUT _ilm/policy/delete_after_1_day
{
  "policy": {
    "phases": {
      "delete": {
        "min_age": "1d",
        "actions": {
          "delete": {}
        }
      }
    }
  }
}
```

> 设置索引模板，引用这个策略


```bash
# 设置索引模板引用这个策略
PUT _index_template/tc_data_center
{
  "index_patterns": ["tc_data_center*"],   // 你希望此模板应用到的索引模式
  "template": {
    "settings": {
      "index": {
        "lifecycle": {
          "name": "delete_after_1_day"  // 引用前面创建的 ILM 策略
        }
      }
    }
  }
}
```
